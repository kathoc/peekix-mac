# Peekix

Mac (Apple Silicon) 専用のシンプルな RTSP ビューアです。ネットワークカメラの映像を低 CPU で安定して表示し続けることに特化しています。

## 特長

- **RTSP 専用ビューア**: H.264 / H.265 のライブ映像をハードウェアデコードで表示
- **しつこい自動再接続**: 通信が途切れても指数バックオフで自動復帰、Wi-Fi 切替やスリープ復帰にも追従
- **省エネ＆カメラ保護**: スリープ・画面ロック・スクリーンセーバ・ディスプレイスリープ・ウィンドウ非表示を検知して RTSP セッションを確実にクローズ。共有カメラ（Eufy 等）の同時視聴が安定
- **常に最前面 / フルスクリーン**: 用途に合わせてウィンドウ表示を切替
- **スクリーンショット**: 表示中のフレームを 1 キーで元の解像度のまま PNG 保存（保存先は環境設定で変更可）
- **軽量**: アプリ本体は 5MB 以下、常駐サービスなし
- **自動アップデート**: Sparkle 2 による署名済みアップデート（既定で1日1回チェック、適用はユーザー確認）

## 動作環境

- macOS 13 Ventura 以降
- Apple Silicon (M1 / M2 / M3 / M4)

## インストール

1. [Releases](https://github.com/kathoc/peekix-mac/releases) ページから最新の `Peekix.app.zip` をダウンロード
2. zip を展開して `Peekix.app` を `アプリケーション` フォルダへドラッグ
3. 初回起動時に「開発元未確認」と表示された場合は、`Peekix.app` を右クリック → `開く` を選択

## 使い方

1. アプリを起動
2. メニューバーの `Peekix` → `環境設定...`（または `⌘ ,`）を開く
3. `RTSP URL` 欄にカメラのアドレスを入力
   - 例: `rtsp://user:pass@192.168.1.10:554/stream1`
4. 環境設定を閉じると自動的に接続を開始

入力した URL は次回起動時に自動で復元されます。

## コントロールバー

ウィンドウ下部のバーから以下の操作ができます。

- 再生 / 停止
- 音声ミュート
- 最前面に固定（ピン）
- フルスクリーン
- スクリーンショット保存
- 接続状態インジケーター（灰: 停止 / 黄: 接続中 / 緑: 再生中）

## キーボードショートカット

| キー | 動作 |
| --- | --- |
| `Space` | 再生 / 停止 |
| `F` | フルスクリーン切替 |
| `V` | ミュート切替 |
| `T` | 常に最前面 切替 |
| `C` | スクリーンショット保存 |
| `⌘ ,` | 環境設定を開く |
| `⌘ ⌃ F` | macOS ネイティブのフルスクリーン |

## 環境設定

- **RTSP URL**: 接続先のカメラアドレス
- **スクリーンショット保存先**: 任意のフォルダを選択可能（既定は `~/Pictures`）

## アップデート

- メニュー `Peekix → アップデートを確認…` でいつでも手動チェック
- 起動後と1日1回のバックグラウンドチェック
- 配布物は EdDSA 署名で改ざん検証

## プライバシー

- テレメトリ・解析は一切ありません
- 通信先は設定した RTSP ホストとアップデートチェック先 (GitHub) のみ

## 変更履歴

### 0.4.1 (2026-04-28)

- 自動アップデートのパッケージング不具合を修正。zip 内のシンボリックリンク欠落とリソースフォーク混入により 0.3.0 → 0.4.0 の自動アップデートが失敗していた問題を解消
- リリース用スクリプト (`Scripts/sparkle-release.sh`) を整備し、`ditto` ベースで再現性のある zip 生成と Sparkle 署名を一括実行

### 0.4.0 (2026-04-28)

- コントロールバーをウィンドウ下部のオーバーレイに変更。マウスホバーでフェードイン、離れるとフェードアウト
- 接続状態インジケーター（ステータスドット）はバーの表示状態に関わらず常時表示
- 動画下のレターボックス領域を廃止し、フルスクリーンを含めて常に映像を最大化
- ダウンスケール時のジャギーを軽減するため Metal レンダラにミップマップと異方性フィルタリングを追加

### 0.3.0 (2026-04-27)

- Sparkle 2 による自動アップデート機能を追加（メニュー「アップデートを確認…」、1日1回バックグラウンドチェック）
- EdDSA 署名によるアップデート検証

### 0.2.0 (2026-04-27)

- スリープ／画面ロック／ディスプレイスリープ／ウィンドウ非表示を検知して RTSP セッションを確実にクローズ
- 共有カメラ（Eufy 等）の同時視聴での復帰安定性を改善

### 0.1.0 (2026-04-26)

- 初回リリース
