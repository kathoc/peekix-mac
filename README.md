# Peekix

Mac (Apple Silicon) 専用のシンプルな RTSP ビューアです。ネットワークカメラの映像を低 CPU で安定して表示し続けることに特化しています。

## 特長

- **RTSP 専用ビューア**: H.264 / H.265 のライブ映像をハードウェアデコードで表示
- **しつこい自動再接続**: 通信が途切れても指数バックオフで自動復帰、Wi-Fi 切替やスリープ復帰にも追従
- **3 つのウィンドウモード**:
  - 通常ウィンドウ
  - フルスクリーン
  - 常に最前面のプレイヤー
- **スクリーンショット**: 表示中のフレームを 1 キーで保存
- **軽量**: アプリ本体は 5MB 以下、常駐サービスなし

## 動作環境

- macOS 13 Ventura 以降
- Apple Silicon (M1 / M2 / M3 / M4)

## インストール

1. [Releases](https://github.com/kathoc/peekix-mac/releases) ページから最新の `Peekix.app.zip` をダウンロード
2. zip を展開して `Peekix.app` を `アプリケーション` フォルダへドラッグ
3. 初回起動時に「開発元未確認」と表示された場合は、`Peekix.app` を右クリック → `開く` を選択

## 使い方

1. アプリを起動
2. 上部の URL 欄に RTSP アドレスを入力
   - 例: `rtsp://user:pass@192.168.1.10:554/stream1`
3. `接続` ボタンを押して再生開始

入力した URL は次回起動時に自動で復元されます。

## キーボードショートカット

| キー | 動作 |
| --- | --- |
| `F` | フルスクリーン切替 |
| `Space` | 再生 / 一時停止 |
| `V` | ミュート切替 |
| `M` | ミニプレイヤー切替 |
| `T` | 常に最前面 切替 |
| `C` | スクリーンショット保存 |
| `⌘ + ⌃ + F` | macOS ネイティブのフルスクリーン |

## プライバシー

- テレメトリ・解析・自動アップデートは一切ありません
- 通信先は設定した RTSP ホストのみ
