# Project Brief: Peekix

## One-line description

Peekix is a lightweight RTSP camera peeking app.

## Product concept

The user should be able to open Peekix and immediately see a security camera feed.

The app should feel closer to “checking a small window” than “operating surveillance software.” It should be light, quiet, persistent, and quick to recover.

## Primary user story

As a user, I want to set an RTSP camera URL once, then open the app whenever I want to glance at the camera feed, without re-entering settings or manually reconnecting.

## Secondary user story

As a user, I want to leave the app open for many hours or days, including across sleep/wake cycles, and have it reconnect automatically when the computer wakes or the network comes back.

## Non-goals

Do not build these in the first version:

- Multi-camera dashboard
- Recording
- Motion detection
- AI detection
- Cloud sync
- User accounts
- Timeline playback
- Remote access service
- Camera discovery
- Complex layout management

## Product principles

- One camera first.
- Show video fast.
- Recover without asking.
- Keep UI quiet.
- Prefer reliability over cleverness.
- Prefer fewer features over unclear behavior.
